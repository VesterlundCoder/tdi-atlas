## 4. Combining Marks (Applied to Base Letters)

These are **zero-width** characters placed after the base letter.

| Mark | Codepoint | Name | Example | Use |
|------|-----------|------|---------|-----|
| ̄ | U+0304 | COMBINING MACRON | ᾱ | Long vowel mark |
| ̅ | U+0305 | COMBINING OVERLINE | κ̅ | **Nomina sacra, numeral markers, abbreviations** |
| ͞ | U+035E | COMBINING DOUBLE MACRON | — | Spanning abbreviation mark |
| ̃ | U+0303 | COMBINING TILDE | — | Scribal variant of perispomeni |
| ͂ | U+0342 | COMBINING GREEK PERISPOMENI | ᾶ | Standard perispomeni (preferred over U+0303 for Greek) |
| ̓ | U+0313 | COMBINING COMMA ABOVE | ἀ | Smooth breathing (psili) |
| ̔ | U+0314 | COMBINING REVERSED COMMA ABOVE | ἁ | Rough breathing (dasia) |
| ͅ | U+0345 | COMBINING GREEK YPOGEGRAMMENI | ᾳ | Iota subscript |
| ̈ | U+0308 | COMBINING DIAERESIS | ϊ | Diaeresis (dialytika) |
| ́ | U+0301 | COMBINING ACUTE ACCENT | — | Use precomposed forms from Greek Extended when possible |
| ̀ | U+0300 | COMBINING GRAVE ACCENT | — | Use precomposed forms when possible |
| ̆ | U+0306 | COMBINING BREVE | — | Short vowel mark |
| ʼ | U+02BC | MODIFIER LETTER APOSTROPHE | — | Elision |

> **Normalisation rule:** Always use **Unicode NFC** (`unicodedata.normalize("NFC", text)`). The Greek Extended block (U+1F00–U+1FFF) provides precomposed forms for all common vowel+diacritic combinations — prefer these over base letter + combining marks.

### Abbreviation Overline — Critical for Nomina Sacra & Numerals

**`κ̅ς̅`** = κ + U+0305 + ς + U+0305 = κύριος (abbreviated)

The combining overline U+0305 is the standard diplomatic encoding for the horizontal bar written over abbreviated words in Byzantine manuscripts.

## 10. Python Normalization Snippet

```python
import unicodedata

def normalize_greek(text: str) -> str:
    """NFC normalization — required for Byzantine polytonic Greek."""
    return unicodedata.normalize("NFC", text)

def to_nfd(text: str) -> str:
    """NFD — for analysis / combining mark manipulation."""
    return unicodedata.normalize("NFD", text)

# Check if a character is Greek
def is_greek(ch: str) -> bool:
    name = unicodedata.name(ch, "")
    return "GREEK" in name

# Detect combining overline (abbreviation mark)
OVERLINE = "\u0305"
def has_overline(ch: str, text: str, pos: int) -> bool:
    return pos + 1 < len(text) and text[pos+1] == OVERLINE

# Detect Milesian numeral (letter + overline)
NUMERAL_MAP = {
    "α": 1, "β": 2, "γ": 3, "δ": 4, "ε": 5,
    "ϛ": 6, "ζ": 7, "η": 8, "θ": 9,
    "ι": 10, "κ": 20, "λ": 30, "μ": 40, "ν": 50,
    "ξ": 60, "ο": 70, "π": 80, "ϟ": 90,
    "ρ": 100, "σ": 200, "τ": 300, "υ": 400, "φ": 500,
    "χ": 600, "ψ": 700, "ω": 800, "ϡ": 900,
}
```
